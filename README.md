# Master Thesis - Supplementary Materials

This repository contains supplementary materials for the master thesis: **Design and Evaluation of Digital Instructions with Corrective Feedback for Exoskeleton Donning.**

## Repository Contents

### Data Files

- **Qualitative_Analysis.xlsx**  
  Contains qualitative data analysis, coding schemes, and thematic analysis results

- **Quantitative_Data.xlsx**  
  Contains raw data across both studies and organized data for various statistical tests

### Analysis Scripts

- **RStudio.zip**  
  Contains two R scripts for computing correlation matrices: one for general correlations and one for device-specific correlations (Apogee and BB devices)

### Study Materials

- **Study_Script.pdf**  
  Contains the study script used during data collection across both studies
